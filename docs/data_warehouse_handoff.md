# Data Warehouse Scope Note

The current reduced class-project version does not implement a data warehouse.

Earlier planning discussed analytical tables, movement facts, dimensions,
aggregates, ETL, and snapshot logic. Those ideas are removed from the current
transactional scope so the project stays small enough for the assignment.

The current database only includes transactional tables for locations,
products, inventory, employees, requests, and shipments, plus three simple
views.
